## The Hidden Cost of Acceptable Medical Device Field Failure Rates

With sustainability being a big talking point in the MedTech
world right now its important that devices that are failing are not just
accepted as 'the norm' or 'just the cost of doing business'. If a Field Service
Engineer (FSE) is going onsite to fix a problem, then you need to understand
why that problem occurred. 

It could simply be premature failure, or a simple user error
which has caused the device to fail in which case you probably don't need to do
anything about it. However, if you have multiple FSE's visiting multiple
customers and they are all seeing the same part failing, that's not something
you can ignore. That is a design issue that needs to be investigated. 

Sending your FSE's out to the same problem repeatedly will
bleed your margins. Where as if you carry out proper Root Cause Analysis (RCA) then you are much more likely to highlight a design failure or a component that is not right for the task at hand. This information can then be fed back into the R&D cycle to be fixed on the V2.0 of the device. It also allows you to implement a field process to fix the problem on all affected devices.
